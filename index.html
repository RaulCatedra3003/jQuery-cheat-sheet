jQuery Cheat Sheet

Events:

  • HTML document has been loaded:

    Vanilla JavaScipt:

    document.addEventListener("DOMContentLoaded", function() { const content = document.getElementById("vanilla-event__1"); content.innerHTML = <p>Document loaded</p>"; })

    jQuery:

    $(function() { $("#jquery-event__1").append(`<p>Document loaded</p>`); })

  • HTML element has been clicked:

    Vanilla JavaScipt:

    const button1 = document.getElementById("vanilla-event__2-button"); button1.addEventListener("click", function() { const content = document.getElementById("vanilla-event__2"); content.insertAdjacentHTML("beforeend", "<p>Button has been clicked</p>"); })

    jQuery:

    $("#jquery-event__2-button").on("click", function() { $("#jquery-event__2").append(`<p>Button has been clicked</p>`); })

  • HTML element has been double clicked:

    Vanilla JavaScipt:

    const button2 = document.getElementById("vanilla-event__3-button"); button2.addEventListener("dblclick", function() { const content = document.getElementById("vanilla-event__3"); content.insertAdjacentHTML("beforeend", "<p>Button has been doble clicked</p>"); })

    jQuery:

    $("#jquery-event__3-button").on("dblclick", function() { $("#jquery-event__3").append(`<p>Button has been clicked</p>`); })

  • User presses a key:

    Vanilla JavaScipt:

    Press a keyboard key.
    document.addEventListener("keypress", function(e) { const content = document.getElementById("vanilla-event__4"); content.innerHTML = `You have pressed ${e.code}.`; })

    jQuery:

    Press a keyboard key.
    $(document).on("keypress", function(e) { $("#jquery-event__4").text(`You have pressed ${e.code}.`); })

  • User moves the mouse cursor:

    Vanilla JavaScipt:

    Move the mouse cursor.
    window.addEventListener("mousemove", function(e) { const content = document.getElementById("vanilla-event__5"); content.innerHTML = `The mouse is in ${e.clientX}x and ${e.clientY}y position.` })

    jQuery:

    Move the mouse cursor.
    $(window).on("mousemove", function(e) { $("#jquery-event__5").text(`The mouse is in ${e.clientX}x and ${e.clientY}y position.`); })

  • User changes a value of an text input:

    Vanilla JavaScipt:

    Change the input value.

    const input1 = document.getElementById("vanilla-event__6"); input1.addEventListener("input", function() { const content = document.getElementById("vanilla-event__6-content"); const value = input1.value; content.innerHTML = `The new value is ${value}.` })

    jQuery:

    Change the input value.

    $("#jquery-event__6").on("input", function() { $("#jquery-event__6-content").text(`The new value is ${$(this).val()}.`) })

  • An image is loaded:

    Vanilla JavaScipt:

    const img1 = document.getElementById("vanilla-event__7-content"); img1.addEventListener("load", function() { const section = document.getElementById("vanilla-event__7"); section.insertAdjacentHTML("beforeend", `<p>The img has been loaded</p>`) })

    jQuery:

    $("#jquery-event__7-content").on("load", function() { $("#jquery-event__7").append(`<p>The img has been loaded</p>`); })

  • An image fails to load:

    Vanilla JavaScipt:

    const img2 = document.getElementById("vanilla-event__8-content"); img2.addEventListener("error", function() { const section = document.getElementById("vanilla-event__8"); section.insertAdjacentHTML("beforeend", `<p>The img can not loaded.</p>`) })

    jQuery:

    $("#jquery-event__8-content").on("error", function() { $("#jquery-event__8").append(`<p>The img can not loaded.</p>`); })
